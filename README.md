# 🎮 FPS Game

A free, browser-based multiplayer FPS game built with **Three.js** and **JavaScript**.

🌐 Play directly in your browser — no download required.

## Features

- 🔫 Multiplayer FPS
- 🌍 Browser-based
- ⚡ Three.js graphics
- 🤝 Peer-to-Peer networking (WebRTC)
- 🔐 Account system
- 🖥️ Low-poly graphics
- 🚀 Runs on GitHub Pages

## Technologies

- JavaScript
- HTML5
- Three.js
- WebRTC
- Node.js (Account Server)
